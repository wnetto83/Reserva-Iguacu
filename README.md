# Reserva Iguaçu — Guia do Morador

Página informativa do Condomínio Residencial Reserva Iguaçu (Foz do Iguaçu/PR),
consolidando as principais regras e informações da Convenção (2014/2016) e do
Regimento Interno (2024) em linguagem acessível, com referência legal em cada item.

## Conteúdo

- 12 seções navegáveis (Construção, Áreas de Lazer, Convivência, Segurança, Veículos, Animais, Mudanças, Aluguel por Temporada, Penalidades, Lixo, Taxas, Governança)
- Navegação lateral com scrollspy
- Busca por palavra-chave com destaque (highlight)
- Layout responsivo (mobile-first)
- Sem dependências externas — HTML/CSS/JS estático

## Estrutura

```
site/
├── index.html
├── styles.css
└── script.js
```

## Deploy

Projeto estático — qualquer hospedagem (Vercel, Netlify, GitHub Pages, etc.).

## Observação

Em caso de divergência entre documentos, prevalece o **Regimento Interno** (mais recente).
Página de caráter informativo — em caso de dúvida, consulte os documentos originais ou a Administração.
